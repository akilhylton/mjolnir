# mjolnir
Hardware keylogger based around the ATtiny85 with the micronucleus firmware. 

## Usage 
```
cd ~/mjolnir/firmware && ./flash_mjolnir.sh
```

Directory Structure
------
    .
        
    ├── Mjolnir-PCB         # Eagle CAD PCB files
    ├── examples            # The example code 
    └── firmware            # Firmware for ATtiny85
